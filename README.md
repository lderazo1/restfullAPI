EJECUCIÓN
---
1. Clonar repositorio
2. Realizar instalación de requerimientos mediante PIP **pip install -r requirements.txt**
3. Ejecutar archivo de configuraciones iniciales **. configuration.sh**
4. Ejecutar Flask **flask run**
